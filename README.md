# What's Your True Main?

A League-inspired personality quiz built with Next.js, TypeScript, Tailwind CSS, and Framer Motion.

It leans hard into dark fantasy event-page energy: dramatic reveal screens, premium game-UI framing, archetype verdicts, fake stat bars, rotating roasts, and a shareable result card.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- html-to-image

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.
